How to join 2 columns in a data set to make a separate column in SQL

sqldatabase

I do not know the code for it as I am a beginner. As an example what I need to do is like there is a column 1 containing First name and column 2 consists of last name I want a column to combine both first name and last name.

Could someone please help me with this?

Rahul Kapoor

asked 2 hours ago

1 Answer

15

You can use the CONCAT function or the || operator to combine columns in SQL:

SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM your_table;

Or alternatively:

SELECT first_name || ' ' || last_name AS full_name FROM your_table;
